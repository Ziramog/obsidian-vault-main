# Refero MCP — AI Agent Integration

> "It studies before it builds — and the output looks designed, not generated."

**URL:** https://refero.design/mcp

## What It Does

Refero exposes a MCP (Model Context Protocol) server that lets AI agents browse and search the curated design library programmatically.

**Use case:** When building UI, an AI agent can query Refero to see how real products solve the same problem — then design with reference, not from scratch.
