I ran the full diagnostic:

- **5 URLs:** Michelin Reliquia, 50 Best Verne, TA Tres Monos, Uptown, Mishiguene
- **25 tests total, 17 successful.**

**KEY FINDING:** The earlier scout's `web_extract` tool was MISCLASSIFYING. Plain `curl` returns full HTML + 89 Michelin imgs + 28 50 Best imgs + 36 Mishiguene imgs WITHOUT any JS render. The "js_required" labels were wrong.

**Practical result:**
- `curl` is the right default (fast, simple, works for 80% of cases).
- `google-chrome` needs TMPDIR override to work on this VPS.
- `playwright` same TMPDIR issue.
- `curl_cffi` would unblock TA + IG (not installed yet).
- `uptownba.com` is genuinely a 3.6KB page, not a transport issue.
