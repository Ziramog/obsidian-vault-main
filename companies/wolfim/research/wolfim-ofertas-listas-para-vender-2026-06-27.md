# Wolfim — ofertas comerciales listas para vender

## Posicionamiento base

Wolfim no debería vender “páginas web”.

La frase correcta es:

> **Te construimos un activo digital para vender mejor: web, catálogo o landing con CTA claro; y si hace falta, le llevamos tráfico con Meta Ads.**

## Regla comercial

La oferta se vende por **vertical + resultado esperado**, no por tecnología.

- No vender Next.js, CMS, base de datos o Medusa como protagonista.
- Vender claridad, consultas, visitas, catálogo, confianza y control comercial.
- La tecnología se menciona solo si el cliente pregunta o si justifica mantenimiento.

# 1. Web + Ads para loteos / desarrollos

## Nombre comercial sugerido
**Loteo Activo — Landing + campaña Meta**

## Para quién es
- loteos
- desarrollos de tierra
- barrios abiertos/cerrados
- fracciones
- proyectos inmobiliarios específicos
- dueños directos con tierra para vender

## Problema que ataca
El desarrollador tiene un activo valioso, pero lo muestra mal o depende de publicaciones sueltas.
La gente no entiende rápido:
- dónde queda
- qué lo hace diferente
- si tiene papeles
- si se puede visitar
- si hay financiación

## Promesa
> Armamos una landing premium del desarrollo y llevamos interesados a una CTA concreta: agendar visita, consultar por inversión o hablar por WhatsApp.

## Qué incluye
- landing premium del desarrollo
- relato comercial del proyecto
- galería visual
- mapa / ubicación si aplica
- beneficios clave: títulos, servicios, acceso, financiación, entorno
- CTA a WhatsApp / formulario / visita
- instalación de medición básica
- campaña Meta inicial dirigida a la zona/interés definido

## CTA recomendado
Principal:
- **Agendar visita al predio**

Secundario:
- **Consultar financiación**
- **Hablar con un asesor**

## Precio sugerido
### Setup web
- desde **USD 450** si es landing premium simple
- desde **USD 650–900** si requiere estructura visual más fuerte, copy, galería extensa o secciones extra

### Mantenimiento
- desde **USD 39/mes**

### Gestión de Ads
- presupuesto publicitario pagado por el cliente
- fee de gestión separado o incluido en un piloto corto
- no absorber gasto publicitario desde Wolfim

## Argumento de venta
> Hoy el problema no es solo publicar el lote. El problema es convertir interés en visita. Para eso necesitás una página clara, visual y con un llamado a la acción directo.

## Caso base
- La Montaña
- Lotes Río Tercero — primera prueba Meta Ads con tráfico a web + CTA

# 2. Web + Ads para autos / agencias

## Nombre comercial sugerido
**AutoLead — ficha/catálogo + campaña Meta**

- vendedores independientes

## Problema que ataca
El vendedor depende de publicaciones dispersas en redes o marketplaces.
No tiene un destino propio donde mostrar mejor el vehículo, captar consultas y medir interés.

## Qué incluye
- ficha individual de vehículo o catálogo simple
- fotos, datos técnicos, precio y financiación si aplica
- CTA directo a WhatsApp
- copy comercial claro
- medición de visitas
- campaña Meta inicial

## CTA recomendado
Principal:
- **Consultar por WhatsApp**

Secundario:
- **Agendar visita / prueba**
- **Pedir financiación**

## Precio sugerido
### Ficha individual / landing de unidad
- desde **USD 200–300**

### Catálogo de agencia
- desde **USD 450**

### Mantenimiento
- desde **USD 25–39/mes** según si es landing simple o catálogo

### Gestión de Ads
- presupuesto publicitario pagado por el cliente
- fee de gestión separado o piloto inicial

## Argumento de venta
> Si el anuncio despierta interés, la ficha tiene que cerrar confianza rápido: fotos, datos, WhatsApp y una consulta sin fricción.

## Caso base
- S&P Cars

# 3. Portal para inmobiliarias

## Nombre comercial sugerido
**InmoPortal Wolfim — catálogo inmobiliario con CMS**

## Para quién es
- inmobiliarias locales
- inmobiliarias familiares
- estudios inmobiliarios premium
- brokers con cartera propia
- equipos que quieren dejar de depender solo de portales externos

## Problema que ataca
La inmobiliaria tiene propiedades, pero su web no transmite confianza o no ordena bien las consultas.
Además depende demasiado de portales externos, redes o WhatsApp manual.

## Promesa
> Te armamos un portal inmobiliario propio para mostrar propiedades, ordenar consultas y elevar la percepción de tu marca.

## Qué incluye
- home institucional
- catálogo de propiedades
- fichas individuales
- filtros / buscador según alcance
- WhatsApp por propiedad
- formulario de contacto
- secciones de confianza: historia, reseñas, equipo, zonas
- CMS o sistema actualizable según necesidad
- SEO base por fichas/secciones

## CTA recomendado
Para compradores:
- **Consultar por esta propiedad**

Para propietarios:
- **Solicitar tasación**

Para inversores:
- **Ver oportunidades / hablar con asesor**

## Precio sugerido
### Portal inmobiliario base
- desde **USD 450**

### Portal con CMS / filtros más completos / carga estructurada
- desde **USD 650–1.200** según alcance

### Mantenimiento
- desde **USD 39/mes**

## Argumento de venta
> Una inmobiliaria no necesita solo una web linda. Necesita un catálogo claro, fichas consultables y una marca que genere confianza antes del primer WhatsApp.

## Caso base
- Roggero & Roma

# 5. Ecommerce con Medusa

## Nombre comercial sugerido
**Tienda Wolfim — ecommerce sobre Medusa**

## Estado comercial
Capacidad disponible, pero todavía sin caso público validado.
No debe ser la primera oferta de salida salvo que el cliente ya tenga fit real.

## Para quién es
- negocio con productos claros
- stock organizado
- capacidad de despacho o retiro
- necesidad real de vender online

## Promesa
> Te armamos una tienda online moderna y administrable para vender con catálogo, carrito y checkout.

## Qué incluye según alcance
- catálogo de productos
- carrito
- checkout
- panel de gestión
- integración de pagos/envíos si aplica
- diseño frontal con marca Wolfim

## Precio sugerido
- desde **USD 450** para implementación base
- escalar según integraciones, cantidad de productos y operación
- mantenimiento desde **USD 59/mes**

## Riesgo
No vender ecommerce a quien todavía no tiene operación lista.
Si no tiene productos, precios, logística o método de cobro definido, primero necesita landing/catálogo, no ecommerce.

# Pitch corto para Juan

## Versión general
> En Wolfim no hacemos solo páginas web. Armamos activos digitales para que tu negocio venda mejor: landing, catálogo, portal o tienda, con llamados a la acción claros y posibilidad de llevar tráfico pago desde Meta.

## Para loteos
> Te armamos una landing premium para mostrar el desarrollo, explicar sus diferenciales y convertir interesados en visitas al predio.

## Para inmobiliarias
> Te armamos un portal propio para ordenar propiedades, mejorar la confianza de tu marca y captar consultas por cada ficha.

## Para autos
> Te armamos una ficha o catálogo online para mostrar vehículos con más confianza y llevar consultas directo a WhatsApp.

## Para PyMEs
> Te armamos una presencia profesional para que cuando te busquen, entiendan rápido qué ofrecés y puedan consultarte sin fricción.

# Regla de cierre

Cuando Juan esté frente a un prospecto, no abrir cinco opciones.
Primero clasificar:

1. ¿Tiene producto puntual para vender? → landing/ficha
2. ¿Tiene catálogo/inventario? → catálogo/CMS
3. ¿Tiene desarrollo/loteo? → landing premium + visita
4. ¿Tiene empresa sin buena presencia? → sitio profesional
5. ¿Tiene operación online lista? → ecommerce

Después ofrecer **un solo camino recomendado**.
