# Wolfim — marco comercial por vertical (jun 2026)

## Tesis central

Wolfim ya no debería venderse como “hacemos webs”.
La estructura comercial correcta hoy es:

> **Wolfim construye activos digitales comerciales y, cuando aplica, los conecta con tráfico pago y CTA.**

Eso permite vender desde una landing simple hasta soluciones con CMS, base de datos y ecommerce.

## Stack de oferta real

### Nivel 1 — entrada
- landing simple
- sitio institucional
- rediseño de web vieja

### Nivel 2 — comercial
- catálogo digital
- ficha comercial por producto/vehículo/propiedad
- landing premium de captación
- sitio con CTA fuerte (WhatsApp / formulario / visita)

### Nivel 3 — estructura
- CMS actualizable
- base de datos
- filtros / buscador
- portales por vertical

### Nivel 4 — crecimiento
- Meta Ads hacia web con CTA
- SEO
- remarketing futuro
- ecommerce sobre template Medusa

## Verticales priorizados

### 🥇 Inmobiliarias
**Oferta principal:** portal con catálogo / CMS

**Problema que resuelve:**
- web desordenada
- propiedades mal presentadas
- consultas sin estructura
- poca confianza visual

**Producto Wolfim:**
- portal inmobiliario
- filtros, fichas, buscador
- WhatsApp por propiedad
- sitio premium + reseñas + mapas

**Caso base:** Roggero & Roma

---

### 🥇 Loteos / desarrollos de tierra
**Oferta principal:** landing premium orientada a visita

**Problema que resuelve:**
- proyecto difícil de explicar
- terreno mal presentado
- falta de relato visual/comercial
- poca claridad para convertir interés en visita

**Producto Wolfim:**
- landing visual premium
- storytelling del proyecto
- galería inmersiva
- CTA para agendar visita
- CTA para inversión
- soporte a campaña Meta

**Caso base:** La Montaña

---

### 🥈 Automotor / agencias de autos
**Oferta principal:** catálogo o landing de vehículo/inventario

**Problema que resuelve:**
- inventario mal mostrado
- anuncios dispersos
- dependencia de marketplaces externos
- consultas sin embudo propio

**Producto Wolfim:**
- fichas de vehículos
- catálogo / inventario online
- CTA a WhatsApp
- tráfico pago a ficha o landing

**Casos base:** S&P Cars + prueba Meta Ranger 2015

---

### 🥉 PyMEs y profesionales
**Oferta principal:** landing o sitio institucional

**Problema que resuelve:**
- presencia floja
- web vieja
- mala percepción de marca
- pocas consultas claras

**Producto Wolfim:**
- landing profesional
- sitio institucional
- SEO base
- CTA por WhatsApp y formulario
- potencial tráfico Meta

---

### 🏅 Ecommerce
**Oferta principal:** tienda online sobre Medusa

**Estado:** capacidad disponible pero todavía sin caso público validado

**Uso recomendado:**
- no empujarlo como primera bandera comercial
- sí usarlo cuando el prospecto ya tiene fit real para venta online

## Productos paquetizables

| Producto | Qué incluye | Verticales naturales |
|---|---|---|
| Landing de captación | 1 página, CTA, WhatsApp, formulario | PyMEs, loteos, campañas puntuales |
| Sitio institucional | varias páginas, marca, servicios, contacto | PyMEs, constructoras, clínicas, estudios |
| Catálogo digital | fichas, buscador, filtros, WhatsApp | inmobiliarias, autos, maquinaria, repuestos |
| Landing premium de desarrollo | storytelling + galería + visita + inversión | loteos, desarrollos, real estate premium |
| Sitio + Ads | activo web + campaña Meta + medición básica | loteos, autos, PyMEs |
| Ecommerce | tienda, checkout, catálogo, backend | retail con fit de venta online |

## Qué vender primero

### Prioridad comercial
1. **Rediseño / reconstrucción web**
2. **Catálogo digital**
3. **Landing premium para loteos**
4. **Sitio + Meta Ads**
5. **Ecommerce** solo en casos con fit claro

## Narrativa comercial sugerida

### Lo que NO conviene decir
- “hacemos páginas web”
- “desarrollo web” genérico
- “te armamos algo moderno”

### Lo que sí conviene decir
- “te armamos un activo digital para vender mejor”
- “ordenamos tu presentación y tus consultas”
- “te llevamos tráfico a una web con CTA”
- “te construimos el catálogo / landing / portal según cómo vendés”

## Oferta nueva que ya queda habilitada

### Web + Ads para loteos
- landing premium
- CTA de visita
- campaña Meta
- seguimiento de visitas

### Web + Ads para autos
- ficha o landing del vehículo / inventario
- CTA a WhatsApp
- campaña Meta
- medición de visitas y consultas

### Web + CTA para PyMEs
- rediseño o landing
- contacto claro
- posibilidad de paid traffic después

## Riesgo a evitar

No vender todo como si fuera lo mismo.
La fuerza comercial aparece cuando Wolfim separa bien:
- portal inmobiliario
- landing de loteo
- catálogo automotor
- institucional PyME
- ecommerce

## Cierre

La estructura comercial correcta para Wolfim hoy es:
- **producto por vertical**
- **CTA claro por caso**
- **web como activo comercial**
- **ads como acelerador cuando hay fit**
