# Farias & Asociados
## Un portal inmobiliario que ayuda a vender mejor cada propiedad

**Para:** Luis Farias

La calidad visual que viste en La Montaña puede ser el punto de partida. Para Farias & Asociados, la propuesta es ir un paso más allá: un portal inmobiliario propio para mostrar propiedades, responder consultas con contexto y trabajar más profesional desde el celular.

No se trata de sumar una web más. Se trata de que cada inmueble tenga una presentación a la altura de lo que Farias quiere vender.

## Lo que cambia para Farias

- Cada propiedad tiene su propia ficha, link y presentación profesional para compartir por WhatsApp.
- El interesado puede filtrar por lo que busca antes de consultar.
- La consulta llega identificando la propiedad, no como un mensaje genérico.
- El equipo puede publicar, actualizar o retirar inmuebles sin depender de terceros para cada cambio.
- Cuando un interesado pide opciones, Farias puede enviar una propuesta PDF ordenada desde el celular.
- Las reseñas ayudan a generar confianza antes de la primera conversación.

## El portal que recibe Farias

### Propiedades listas para compartir

Un catálogo claro, con fotos, precio, ubicación, características y propiedades destacadas. Cada ficha incluye galería, descripción, datos del inmueble, mapa o zona aproximada y contacto directo.

**Resultado:** cuando aparece una consulta, se comparte una propiedad puntual con un link propio. No hace falta mandar fotos sueltas ni buscar publicaciones viejas.

### Búsqueda rápida para el interesado

Filtros por tipo de propiedad, operación, zona, precio, dormitorios, baños y superficie.

**Resultado:** la persona encuentra más rápido lo que necesita y llega a una consulta más calificada.

### WhatsApp con contexto

Cada propiedad tiene un botón de WhatsApp que abre un mensaje listo para enviar con referencia al inmueble consultado.

**Resultado:** menos idas y vueltas para entender qué propiedad está mirando cada persona.

### Panel privado para manejar el inventario

Acceso privado para cargar, editar, publicar, despublicar o destacar propiedades. Incluye gestión de fotos y categorías como casas, departamentos, terrenos y comerciales.

**Resultado:** el portal se mantiene actualizado y Farias conserva el control del catálogo.

### Propuestas PDF desde el celular

Cuando un cliente pide alternativas, se pueden seleccionar propiedades, sumar datos del interesado, agregar condiciones comerciales y generar un PDF profesional para compartir por WhatsApp.

**Resultado:** una consulta pasa de ser un chat informal a una propuesta clara, con marca propia y seguimiento ordenado.

### Reseñas que generan confianza

Un módulo para cargar reseñas reales, elegir cuáles mostrar y destacar las que mejor representan el servicio de Farias.

**Resultado:** compradores, vendedores e inversores encuentran prueba social antes de escribir.
