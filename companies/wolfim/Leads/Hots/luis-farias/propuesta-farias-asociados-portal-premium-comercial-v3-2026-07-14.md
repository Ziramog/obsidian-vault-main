# Farias & Asociados
## Un portal propio para vender mejor cada propiedad

**Para:** Luis Farias

La calidad visual que viste en nuestros demos es el punto de partida. Para Farias & Asociados, la propuesta es un portal inmobiliario propio para mostrar propiedades, responder consultas con contexto y trabajar con una presentación profesional en cada etapa.

No es sumar una web más. Es tener una herramienta comercial que ordena el catálogo, mejora cómo se comparte cada inmueble y acompaña una consulta desde el primer mensaje hasta una propuesta concreta.

## Lo que cambia para Farias

### Cada propiedad se presenta como corresponde

Cada inmueble tiene una ficha propia, con fotos, ubicación, características, precio, descripción y un link profesional para compartir por WhatsApp.

Cuando alguien consulta, no hace falta buscar fotos o publicaciones viejas: se comparte la propiedad exacta, con toda la información en un solo lugar.

### La consulta llega con contexto

El interesado puede buscar por tipo de propiedad, operación, zona, precio, dormitorios, baños o superficie. Desde cada ficha, el botón de WhatsApp identifica la propiedad consultada.

Eso evita mensajes genéricos y permite responder más rápido desde el primer contacto.

### Inventario vivo

Las propiedades entran, cambian y se comparten todos los días. Una alta, una modificación de precio o una oportunidad para destacar se actualizan sin frenar el ritmo de la inmobiliaria.

Lo que el cliente ve refleja lo que Farias puede ofrecer hoy.

### Cada consulta llega con contexto

La ficha concentra fotos, ubicación, precio y características. Cuando alguien escribe por WhatsApp, Farias sabe exactamente qué propiedad le interesa.

### Alternativas listas para enviar

Si un interesado pide alternativas, Farias puede seleccionar propiedades y enviar una propuesta PDF profesional desde el celular.

La conversación deja de ser un chat desordenado y se transforma en una propuesta con marca propia, lista para revisar y seguir.

### La confianza empieza antes del mensaje

Farias puede elegir, ordenar y destacar reseñas reales para que compradores, vendedores e inversores conozcan cómo trabaja antes de escribir.

## Qué cubre el mantenimiento

### Incluye

- **Hosting del sitio:** el servidor donde vive el portal para que esté online.
- **Dominio:** renovación anual del dominio del portal.
- **Backup automático semanal:** una copia de seguridad para poder restaurar el sitio si ocurre un problema.
- **Retoques sobre la estructura existente:** cambiar un título, modificar un texto, actualizar un teléfono o ajustar una descripción.
- **Soporte por WhatsApp en horario comercial:** si aparece una falla real del sitio, la respuesta llega el mismo día hábil.
- **Reportes mensuales avanzados de Analytics:** seguimiento claro de visitas, consultas y comportamiento relevante para tomar decisiones comerciales.

### Se cotiza aparte

- Cambios de estructura, como agregar secciones, mover bloques o rediseñar partes del sitio.
- Funcionalidades nuevas, como formularios especiales, integraciones con sistemas o carrito.
- Contenido nuevo, como fotos profesionales, redacción extensa o traducciones.
- Google Ads o presupuesto de publicidad.
