## Archivo 2: google-ads-inmobiliaria.html

**Tipo:** Guía operativa/tutorial de Google Ads (dark theme)

### Estructura

- Budget rows, keyword tables, ad previews, timelines
- Script de conversación agente/cliente

### Contenido técnico
- Configuración paso a paso de cuenta Google Ads
- Estructura de campañas (compradores, alquileres)
- Keywords + match types + negativas
- 15 títulos + 4 descripciones para RSA
- Extensiones (sitelinks, call, location, snippets)
- Estrategia de puja por etapas (CPC manual → Maximizar clics → Maximizar conversiones → CPA objetivo)
- Revisión semanal y mensual
- Métricas objetivo: CTR >3%, Quality Score ≥7, IS >50%, CPL rastreable

### Presupuesto Córdoba 2025 (en pesos)
- Mínimo: $150-250k ARS/mes
- Recomendado: $400-700k ARS/mes
- Ideal: $1M+ ARS/mes
- CPC estimado: $800-2,500 ARS/clic
- Leads esperados: 15-35/mes
- CPL estimado: $12-25k ARS

- **Tema:** Dark (#0E0E0E), dorado (#C9A84C), azul Google

## Lo mejor de ambos — cómo usarlo

### Para propuestas comerciales a clientes:
→ Usar `propuesta-google-ads-base.html` como estructura base
→ Reemplazar data de Roggero por data del cliente
→ Mantener tono editorial premium

### Para referencia operativa interna:
→ Usar `google-ads-inmobiliaria.html` como knowledge base
→ Extraer keywords, negativas, estructura de campañas
→ Para training o cuando no sabés cómo configurar algo
