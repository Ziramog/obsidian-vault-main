# Template Propuesta Wolfim

## Archivo base
`Propuesta_Farias_final.html`

## Uso
- Template HTML para propuestas comerciales Wolfim
- Generar PDF con Puppeteer

## Estructura
- Header oscuro (#1a1a2e) + logo gold (#f5c518)
- Fondo blanco
- Cards con `break-inside: avoid`
- Diagnóstico en rojo (#c62828)
- Solución en verde (#2e7d32)

## Flujo de uso
1. Copiar `Propuesta_Farias_final.html`
2. Renombrar con nombre del cliente
3. Editar datos: nombre, empresa, diagnóstico, propuesta, precio
4. Generar PDF con Puppeteer
