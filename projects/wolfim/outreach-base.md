# Wolfim — Base de Mensajes Outreach

## Secuencia Completa

---

### MENSAJE A — Primer contacto (lead tibio)

```
Hola [Nombre], soy Juan de Wolfim.
Estuve revisando el sitio de [Nombre Inmobiliaria] y tengo un diagnóstico gratis: vi qué está pasando con las consultas y tengo ideas concretas para mejorarlo.
Si te interesa, te paso el PDF con lo que encontré. ¿Te lo mando?
```

**Canal:** WhatsApp
**Timing:** Día 0
**Condición:** Lead tibio — ya mostraste interés o te referenciaron

---

### CONFIRMACION — Si dice "sí, mandalo"

```
Perfecto, te lo paso ahora.
```

**Timing:** Inmediato
**Acción:** Generar PDF con editor.html → enviar

---

### MENSAJE B — 24h después (sin respuesta al PDF)

```
[Nombre], ¿pudiste ver el PDF? Tengo algunas ideas específicas para [Nombre Inmobiliaria] que creo que pueden funcionar.
¿Te parece si te muestro en 2 minutos cómo se aplicaría a tu caso?
```

**Canal:** WhatsApp
**Timing:** 24h después del envío del PDF
**Condición:** No respondió al PDF

---

### MENSAJE C — Seguimiento (3-4 días sin respuesta)

```
Hola [Nombre], ¿tuviste oportunidad de ver el diagnóstico?
Hay algo que quiero contarte que puede marcar la diferencia para [Nombre Inmobiliaria] en las próximas semanas.
¿Hablamos?
```

**Canal:** WhatsApp
**Timing:** Día 3-4 después del mensaje B
**Condición:** Silencio

---

### MENSAJE D — Breakup (7+ días sin respuesta)

```
[Nombre], no te molesto más.
Si en algún momento querés recuperar esas consultas que se escapan, avisame y con gusto te paso las ideas.
Éxitos con la inmobiliaria.
```

**Canal:** WhatsApp
**Timing:** Día 7-10
**Condición:** No hubo respuesta en toda la secuencia

---

## Hooks Complementarios (alternativas)

### Hook B — Más curiosidad

```
Hola [Nombre] 👋 Te contacto desde Wolfim.
Hace poco analicé los sitios de inmobiliarias en [localidad] y hay algo que veo siempre: inmobiliaria con Instagram pero sin web propia pierde consultas todos los días.
Si te parece, te paso un PDF con lo que encontré en tu caso. ¿Te interesa?
```

### Hook C — Ultra corto

```
Hola [Nombre], soy Juan de Wolfim.
Vi el sitio de [Nombre Inmobiliaria] y tengo ideas concretas para atraer más consultas. ¿Te paso el diagnóstico gratis?
```

### Hook D — Con prueba social

```
Hola [Nombre] 👋 Wolfim.
Trabajo con inmobiliarias de [localidad] y tengo tu sitio diagnosticado: qué está funcionando y qué podriamos mejorar para generar más consultas.
¿Te paso el PDF con el análisis?
```

## Objeciones Previsibles (respuestas)

### "No tengo tiempo ahora"

```
Entiendo, está bien. ¿Te parece si te marco en el calendar para la semana que viene? Así no perdés la info.
```

### "Ya tengo web"

```
Perfecto, ¿te está generando las consultas que querés? Porque si no, el problema no es tener web — es que la web trabaje para vos.
```

### "Es muy caro"

```
Totalmente válido. La pregunta es si el costo de NO tener una web que genera consultas es mayor. ¿Querés que te muestre qué consultas se están perdiendo?
```

### "No me interesa"

```
Gracias por la honestidad. Si cambias de opinión en el futuro, aquí estoy.
```

---

## Notas

- **Idioma:** Informal, cercano, sin jerga técnica
- **Nombre del remitente:** Juan (no "Wolfim" como empresa)
- **Extensión máxima:** 3-4 oraciones
- **CTA siempre:** Una sola pregunta, respuesta fácil
- **Timing:** Martes-jueves 9-11am o 17-19pm (horarios laborales)
